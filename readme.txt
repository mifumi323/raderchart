レーダーチャート描画クラス

バージョン：1.0.1
制作者：美文
URL：https://tgws.plus/

【概要】
簡単なレーダーチャートを描画します。組み込んで使います。

多機能・高性能なグラフ描画ライブラリは様々なものが存在しますが、こちらは本当にただ単にレーダーチャートを描画するだけです。
その分、単機能に絞ってあるので導入しやすいと思われます。
単にレーダーチャートを出したいだけならサンプルをちょこっといじるだけ。

文字の描画も一応できますが、その場合はTrueTypeフォントが必要です。

要PHP5+GD2
